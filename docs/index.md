## About Myriad

Myriad is an open-source standardization, functionality, and mitigations test for executors.

After the discontinuation of [UNC](https://www.roblox.com/games/79035306837882), multiple forks and continuations of it have popped up, as well as separate general executor tests such as vulnerability tests and identity tests. Myriad hopes to achieve these as one, unified test.
