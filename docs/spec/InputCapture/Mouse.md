# InputCapture.Mouse

[`InputCapture`]: ./index.md

The descriptor for a mouse based input in the [`InputCapture`] API.

## Members

| Code          | Note                            |
| ------------- | ------------------------------- |
| `Left`        | The left mouse button.          |
| `Right`       | The right mouse button.         |
| `Middle`      | The mouse wheel button.         |
| `ScrollUp`    | The mouse wheel scrolling up.   |
| `ScrollDown`  | The mouse wheel scrolling down. |
| `SideBack`    | The mouse side back button.     |
| `SideForward` | The mouse side forward button.  |
